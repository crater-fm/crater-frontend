How to deploy web server to Heroku:

// First, stage and commit all changes

git add .
git commit -m "message"

// Then login to heroku and set remote

heroku login
heroku git:remote -a heroku-app-name

// Then push code to deploy
git push heroku main

// Check that some dynos are running
heroku ps:scale web=1

// Open in browser
heroku open


To deploy the Crater database to Heroku:
datasette publish heroku mydatabase.db
